## Supported versions & requirements:

- Swift 5+
- iOS 13+
- Xcode 14+

## Features

- Remote Player management
- iOS 16 support
- Google IMA support
- OMSDK support
- Sample player applications & code samples
- Fullscreen playback management
- Fully-featured Player API to access Player events, state and native control
